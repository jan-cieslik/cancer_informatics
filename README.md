# Cancer Informatics

The Cancer Informatics webpage is built using [Docusaurus 2](https://docusaurus.io/), a modern static website generator.

## Installation

- Install NPM: [Download](https://nodejs.org/en/download/)
- Install Docusaurus
```
npm install
```

## Local Development

Run development server:
```
npm run start
```
